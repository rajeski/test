My Flicks, Movie API Documentation

This web application will provide end-users with access to information for a variety of genres, movies and directors. Users accessing this website will be able to sign-up; update their details as well as create a list of their favorite movies.

Request URL HTTP Method Request Body Data Format Response Body Data Format
Display a list of ALL available movies /movies GET None JSON containing ALL available movie data
Display a single movie's data by title /movies/[Title] GET None JSON containing ALL available data about a given movie, including title, description, genre, director and movie-related image. Example:

{ title: 'Winged Migration', genre: 'nature tail', director: 'jacques perrin, jacques cluzard, michael debats', main_actor: 'birds', country: 'world', year: 2003 },

Display movie genre data /movies/genres/[Name] GET None JSON containing data about a movie genre. Example:

{ "Name": "Crime", "Description": "A film genre whereas the protagonist and/or protagonists grapple with ethical or moral choices often involving violence, law enforcement, death and good versus evil." }

Display director data /movies/directors/[Name] GET None JSON containing a given director's bio. Example:

{ "Name": "Stanley Kubrick", "Bio": "Stanley Kubrick was an American film director, screenwriter and producer." "Born": "1928" "Died": "1999" }

Add new user accounts /users POST JSON containing user data, containing the following:

{ "username" : "genericman", "password" : "generic123", "email" : "generic@man.com", "date_of_birth" : "1984-05-01" }

JSON containing user data, including generating a unique My Flicks ID:

{ "topMovies": [], "_id": "Need_to_figure_out_how_to_generate_this" "username" : "genericwoman", "password" : "generic456", "email" : "generic@woman.com", "date_of_birth" : "1989-03-01" "__v": 0 }

Display an user's information update /users/[Username] PUT JSON containing user's data, containing the following:

{ "username" : "genericman", "password" : "generic123", "email" : "generic@man.com", "date_of_birth" : "May 1, 1984" }

Post message confirming user's information was updated
Add movies to user's list of favorites /users/[Username]/Movies/[MovieID] POST None Post message confirming a movie or movies was added to a given user's favorites' list
Remove a movie or movies from a given user's favorites' list /users/[Username]/Movies/[MovieID] DELETE None Post message confirming the movie was removed
Remove a user's accounts /users/[Username] DELETE None Post message confirming the user's account was deleted
